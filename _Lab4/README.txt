# Лабораторная 4.
Эта работа представляет собой windows службу для работы с базой данных.
Я создал базу данных под названием GameDev, состоящую из 5 таблиц : GameIndustry, Games, Companies, Publishers, Developers.
Фотография диаграммы прикреплена и называется DataBase.png.
Класс DataBaseOperations нужен для извлечения данных из бд и помещения их в класс-контейнер GameDevEssence (проект DataAccess)
Проект DataManager содержит класс с интерфейсом для сериализации в XML файл.
DataModel - проект со всеми классами базы данных и классом-контейнером GameDevEssence.
ServiceLayer - собственно сама windows служба, в которой происходит вызов нужных методов для работы с базой данных.
